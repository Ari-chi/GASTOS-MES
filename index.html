Seguí estos pasos para exportar desde PowerBI Desktop:
1Abrí el archivo Comercial Crédito.pbix en PowerBI Desktop
2Andá a la página "DETALLE COMPROBANTES" (segunda pestaña)
3Hacé clic en la tabla → aparecen los ··· arriba a la derecha del visual
4Clic en ··· → Exportar datos → Datos subyacentes → Exportar
5Se descarga un archivo con columnas: MES · TIPO DE CUENTA · IMPORTE. Subilo acá.
✅ Si el archivo tiene columna MES, el HTML carga todos los meses de una sola vez.
